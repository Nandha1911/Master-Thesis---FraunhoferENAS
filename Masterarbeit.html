Aim of the thesis:

In this thesis, a power electronic component is modeled and simulated using the finite element software Abaqus. The simulation results are combined with analytical approaches—specifically, non-linear damage models—to predict the damage progression in the SAC (Sn-Ag-Cu) die attach layer. This methodology enables the evaluation of damage percentage without the need for physical testing, thereby saving time and resources while maintaining accuracy.

  • After applying mesh, boundary conditions, and apply initial thermal load (270°C to 125°C over 3600 s).
  • Initial damage area (7.67%) is removed after the first step.
  • Restart from step 2 and run first simulation for 3 thermal cycles (125°C → −40°C → 125°C).
  • Extract CEEQ values (10% volume) and calculate Δε (step 8 − step 6).
  • Use Coffin-Manson equation with C1 and C2 to estimate cycles to failure.
  • Determine damage value using linear or nonlinear damage models.
  • Identify newly damaged elements and deactivate them in the next simulation.
  • Repeat for 5 simulations (15 thermal cycles total), updating damage and Δε each time.

Physics-Informed Parameter Identification

PINN Architecture
Input: Strain/Temp → Neural Network → Predicted Stress
Loss = MSE(Data) + λ · Loss(Viscoplastic Residuals)

Fig 3.2: PIML framework for material calibration
  • Framework: Developed using PyTorch for the neural network architecture and automatic differentiation (Autograd).
  • Physics Loss: Integrated viscoplastic constitutive equations directly into the loss function to ensure thermodynamic consistency.
  • Optimization: Utilized SciPy (L-BFGS-B) as a second-order optimizer to achieve high-precision convergence of physical parameters.
  • Hybrid Approach: Combined sparse experimental data with high-fidelity Abaqus simulation results to calibrate non-linear damage models.
  • Accuracy: Enabled the identification of HRSu model parameters that match experimental delamination trends with >90% accuracy.
